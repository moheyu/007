个人轻量化通用RAG智能问答系统

(docs/images/cover.png)

面向个人自用的检索增强生成（RAG）对话系统，支持通用网页爬虫采集、本地向量库私有化存储，基于专属知识库实现精准、低幻觉的大模型智能问答，个人电脑即可一键部署运行。

项目简介

本项目是一套开箱即用的个人专属RAG问答工具，核心解决大模型对话幻觉、零散网页/文档知识难以统一管理检索的痛点。你可以通过通用爬虫采集任意合法网页的内容，一键构建本地私有化向量知识库，让大模型基于你采集的专属内容进行对话问答，全程数据本地存储，兼顾便捷性与隐私性。

✨ 核心功能特性

(docs/images/architecture.png)

- 通用网页数据采集：自研轻量化爬虫，支持任意合法站点的递归深度爬取、断点续爬、URL级别去重，自动解析网页正文，输出结构化的JSON/Markdown文本数据，无站点专属绑定，适配各类教程、文档、资讯、博客等网页。
- 轻量化本地向量化：采用 bge-small-zh-v1.5 开源中文嵌入模型，CPU即可流畅运行，无需GPU显卡，零门槛实现文本语义向量化转换。
- RAG检索增强对话：对接阿里云DashScope通义千问大模型，基于检索到的本地知识库原文内容生成精准回答，从根源上降低大模型幻觉，实现专属知识库的定制化问答。
- 全量+增量双模式索引：首次使用支持全量构建向量索引，后续新增内容仅需增量更新，无需重复处理已有数据，大幅提升知识库更新效率，适配个人知识库持续迭代的需求。
- 双模式交互：同时提供「命令行交互式对话」和「HTTP API服务」两种使用模式，既可以本地终端直接问答，也可以通过API对接前端、机器人等二次开发场景。
- 私有化本地部署：极简环境依赖，个人Windows/Mac/Linux电脑均可一键部署，向量库、采集数据全部本地持久化存储，保障个人数据隐私安全。
- 自动化增量更新：内置快照标记与监控机制，爬虫完成后可自动触发增量索引构建，无需人工手动执行，实现知识库的自动化维护。
